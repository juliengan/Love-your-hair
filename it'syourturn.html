Memory game

Hey, let's play the memory game. The goal of this game is to match the pairs of cards 2 by 2. If the cards flipped face up are the same you win, otherwise the cards are automatically turned face down and you need to make a new try. Try to find all the pairs in 50 seconds. Good luck !
